▸ How it works · Three-layer scoring · Trading signals

v7.1 Intraday %Chg column (Intra%): Shows price change vs today's open, separating intraday momentum from pre-market gap. The %Chg column (vs yesterday's close) bakes in both — Intra% isolates the intraday-only move that your strategy is designed to trade. Tooltip on hover shows both values plus today's open price.

v7.1 Gap-relationship icon (G column): ↗↗ gapped up and continuing up · ↘↘ gapped down and continuing down · ↗↘ gap up but fading intraday · ↘↗ gap down but bouncing intraday · ↗━ gap up but flat intraday (stale gap) · ↘━ gap down but flat intraday. The 4 most informative patterns: ↘↗ is your "second low bounce" setup like NVDA Tuesday; ━↗ means fresh intraday strength with no premarket bias.

v7.1 Trend layer scoring update: The Trend layer now uses Intraday %Chg (price vs today's open) instead of overall %Chg for its directional reinforcement component. This prevents stale pre-market gaps from inflating the Trend score for stocks that aren't actually moving intraday.

v7.1 Momentum-type fix: The Type column now persists the last classification when a ticker leaves candidate state. In v7, Type was nulled when the candidacy ended, which is why the session summary showed empty Type fields for all tickers including INTC. Sustained-trend tickers (steady score, low velocity, no flips) now correctly classify as DRIFT after 2+ minutes in candidate state.

v7 Three-Layer Architecture: The single blended score from v6 is replaced by three independent layer scores. Each layer captures a different time horizon and signal type. A ticker is shown in the Candidates section corresponding to its strongest active layer.

TREND Layer (slow, 30min-hours) — Built from VWAP. Captures the "3% in 3 hours" subtle moves. Three sub-components: price-vs-VWAP position (ATR-normalized), VWAP slope direction, and RVol+pctChg confirmation. Smoothing time constant 5 min. Default qualifying threshold 65 with 120s minimum streak. This is the slow, sustained-trend signal.

BREAKOUT Layer (medium, 5-30min) — Range expansion + VWAP band breaks. Built on v6's consolidation breakout detector with VWAP-derived bands and volume confirmation. Smoothing time constant 90s. Default threshold 70 with 30s minimum streak. Fires when consolidation gives way to expansion.

PRESSURE Layer (fast, 30s-5min) — Three OFI proxies blended: (1) average bid-ask position of recent prints over 60s, (2) volume rate vs session average, (3) size-weighted decaying tick clusters with burst detection. Smoothing 30s. Default threshold 72 with 15s minimum streak. Captures aggressive intraday flow.

VWAP Tracking (v7): A session-cumulative VWAP is computed per ticker from price+volume snapshots. The VWAP column shows current price as a percentage relative to VWAP — green when above, red when below. VWAP slope (history over 30 min) feeds the Trend layer.

OFI Proxies (v7): Order Flow Imbalance in the academic sense requires L2 depth-of-book data. On L1, three proxies approximate it: average quote-position over 60s (smooths out per-tick noise that hurt v5), volume-rate ratio (recent vs session average), and size-weighted exponentially-decaying tick clusters with burst detector for institutional sweeps. None is reliable alone; the blend is robust.

Three independent classifications: Each layer has its own qualifying threshold, stability window, minimum streak, and flip count. A ticker's flip count is the sum across all three layers. The Streak column shows the longest active streak across any layer.

Reading the layer score columns: Each cell shows a horizontal bar with the score number, colored in the layer's accent. Brighter color = higher score. A small dot indicator appears when the layer is currently in bull/bear state (passed stability gate).

v7 Accessibility: Default theme is high-contrast (pure black bg, pure white text). Three additional themes available in Settings → Accessibility (medium contrast, original v6 colors, light mode). Base font size adjustable 11px-22px. All UI elements scale together.

Carryover from v5/v6: Bulletproof persistence, time-of-day tuning, regime indicator, momentum-type classifier, Don't-Trade warning, session summary report.
